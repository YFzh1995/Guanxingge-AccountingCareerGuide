## 关于本作品

本课程是作者基于十余年财务职场观察与数百次1v1咨询经验，系统化提炼的职业发展框架。核心视角是**"职场结构主义"**——即个人职业成就由三个变量共同决定：

- **引力场（环境）**：行业、公司、组织的结构性红利与约束
- **个人禀赋（自身）**：能力资产、动力、约束条件
- **结构洞察（判断）**：在特定结构中识别稀缺生态位的能力
