## 关于本作品

本课程是作者基于十余年财务职场观察与数百次1v1咨询经验，系统化提炼的职业发展框架。核心视角是**"职场结构主义"**——即个人职业成就由三个变量共同决定：

- **引力场（环境）**：行业、公司、组织的结构性红利与约束
- **个人禀赋（自身）**：能力资产、动力、约束条件
- **结构洞察（判断）**：在特定结构中识别稀缺生态位的能力

## 目录结构

### 底层逻辑（第1-5讲）
- 第01讲 认知重塑：打破"努力的幻觉"与结构主义视角
- 第02讲 底层逻辑：职场结构主义与核心公式
- 第03讲 宏观场域：如何看懂行业与公司对财务人职业发展的影响
- 第04讲 组织引力：财务架构演变与岗位路径选择
- 第05讲 个人禀赋：财务人职业发展的动力、约束和能力资产

### 演进路径（第6-8讲）
- 第06讲 新人的原始积累（25岁｜0-5年）：职场价值锚定与职业资产构建
- 第07讲 大头兵成长记（30岁｜5-10年）：从线性交付到赛道深耕
- 第08讲 解码财务一号位（35岁｜10-15年）：看清职场终局，选准你的赛道

### 实战与终局（第9-10讲）
- 第09讲 实战案例：结构主义视角下的关键抉择
- 第10讲 终局思考：AI时代的"图钉型"生存指南
